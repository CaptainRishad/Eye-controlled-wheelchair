## ğŸ“Œ Overview

This project uses **IR-based eye blink sensors** mounted on glasses to detect eye movements.
The detected blink patterns are processed by an **Arduino Nano**, which sends control commands wirelessly via **Bluetooth (HC-05)** to an **Arduino Uno** that controls the wheelchair motors using relays.

The system is designed to be:

* Affordable
* Easy to build
* Reliable for assistive mobility applications
